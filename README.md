# 💾 ObjeX
Repositório de projeto da disciplina de REQ-T2.


O ObjeX é uma plataforma web de auxilio ao professor projetada para promover a interatividade e o engajamento dos alunos. Os professores podem criar e gerenciar missões educacionais, oferecendo desafios específicos para os estudantes aprimorarem seus conhecimentos. A plataforma facilita a formação de grupos para colaboração em tarefas e permite o compartilhamento de trabalhos finais entre os alunos, promovendo uma aprendizagem mais colaborativa e envolvente. É uma ferramenta inovadora que visa tornar o processo de aprendizagem mais dinâmico e eficaz.
